# NM

고음질 무료 음악 봇, NM

## 🎵 지원하는 플랫폼

NM은 유튜브를 메인으로 지원하며, 사운드클라우드와 스포티파이 또한 부분적으로 지원해요.

## 🤖🛠️ 사용 가능한 명령어

| 명령어      | 한줄 요약                                                                |
| ----------- | ------------------------------------------------------------------------ |
| `/play`     | 검색어 또는 URL로 음악을 찾고, 단일 곡이나 재생목록을 대기열에 추가해요. |
| `/search`   | 플랫폼을 지정해 음악을 검색하고, 선택한 결과를 대기열에 추가해요.        |
| `/queue`    | 대기열을 확인할 수 있어요. 페이지네이션과 새로고침 기능을 지원해요.      |
| `/now`      | 현재 재생 중인 곡의 세부 정보와 진행 상황을 보여줘요.                    |
| `/stop`     | 플레이어를 정지시키고 음성 채널에서 나가요.                              |
| `/pause`    | 재생 중인 음악을 일시정지해요.                                           |
| `/resume`   | 일시정지된 음악을 다시 재생해요.                                         |
| `/repeat`   | 현재 곡 반복, 대기열 반복, 또는 반복 해제를 설정해요.                    |
| `/volume`   | 볼륨을 0~100% 범위로 설정해요.                                           |
| `/autoplay` | 현재 재생 중인 곡을 기준으로 연관된 곡들을 자동으로 재생해요.            |
| `/clear`    | 현재 대기열에 쌓여 있는 곡들을 모두 비워요.                              |
| `/remove`   | 대기열의 특정 위치에 있는 곡을 제거해요.                                 |
| `/seek`     | 현재 곡을 원하는 시각으로 건너뛰어요.                                    |
| `/shuffle`  | 대기열을 랜덤·역순·라운드 로빈 방식으로 섞어요.                          |
| `/skip`     | 지정한 개수만큼 곡을 건너뛰어요. 기본은 1곡이에요.                       |
| `/speed`    | 재생 속도를 0.5~2.0배 사이에서 조절해요.                                 |
| `/ping`     | 봇 지연시간과 Discord API 지연시간을 측정해요.                           |
| `/info`     | 봇의 버전, 제작자, 서버/유저 수 등 상태 정보를 확인해요.                 |
